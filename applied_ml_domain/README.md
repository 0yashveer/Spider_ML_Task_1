# Instructions

1. Download app.py , index.html, index.py
2. ingest.py helps in ingesting files to the database, which is run once while cold start of the model.
3. once vector collection is made , run ollama pull llama3:8b cmd in your terminal after downloading the mentioned model.
4. run app.py and start asking your doubts.

# Information 

1. embedding model used is all-MiniLM-L6-v2
2. llama3:8b model is used as llm
3. chroma db is used as vector db
